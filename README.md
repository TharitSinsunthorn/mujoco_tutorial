# mujoco_tutorial

## Prerequisties
* Ubuntu 20.04
* ros2-foxy

## Installation
* Installation of Python3: https://www.python.org/downloads/

* Installation of MuJoCo
```bash
pip install mujoco
pip3 install -U 'mujoco-py<2.2,>=2.1'
pip install stable-baselines
pip install stable-baselines3
pip install gym 
```
